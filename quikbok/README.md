# quikbok
> AI-powered booking agent for hotels, restaurants & tour operators

## What it does
Bukio is a SaaS that gives any small business an AI booking agent 
via web chat widget, booking link, or WhatsApp — without needing 
a website or tech knowledge.

## Tech Stack
- Python + Flask (backend)
- OpenAI API (AI brain)
- Supabase (database)
- Twilio (WhatsApp)
- Razorpay (payments)
